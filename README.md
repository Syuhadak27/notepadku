**Aplikasi Notepad Offline Modern dengan IndexedDB**


---

## 🌟 Fitur

### ✨ **Core Features**
- ✅ **100% Offline** - Single HTML file, tidak perlu server atau internet
- ✅ **IndexedDB** - Penyimpanan lokal yang powerful dan persisten
- ✅ **Responsive Design** - Optimal di desktop, tablet, dan mobile
- ✅ **Dark/Light Mode** - Sesuai preferensi sistem atau manual

### 📝 **Editor**
- 📖 **Dual Mode** - Mode Lihat (Read-Only) & Mode Edit
- 💾 **Autosave** - Simpan otomatis setiap 2 detik (configurable)
- 📊 **Word & Character Count** - Realtime statistics
- 🎨 **Customizable** - Font size & font family bisa diatur

### 🔧 **Manajemen Catatan**
- 🔍 **Search** - Cari berdasarkan judul atau isi catatan
- 🔄 **Sort** - Urutkan: Terbaru, Terlama, A-Z
-  **Export** - Download catatan sebagai file .txt
- 📥 **Import/Backup** - Backup & restore semua catatan sebagai JSON

### ⌨️ **Keyboard Shortcuts**
- `Ctrl/Cmd + N` - Buat catatan baru
- `Ctrl/Cmd + S` - Simpan catatan
- `Esc` - Tutup editor/modal
